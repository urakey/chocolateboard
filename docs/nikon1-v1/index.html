Nikon 1 V1

  • Posted on April 12, 2019
  • Categories note
  • Tags misc

SONY α7 III を買ったので、Nikon 1 V1 とお別れした。

見た目が気に入って買ったカメラだったから、売るかどうか迷っていたけれど、使わずに寝かせるのはかわいそうなのでメルカリに出品。
いざ売れると別れ惜しい気持ちになってしまった。

とっても欲しそうなよい人にもらわれていくことになりました。本当に良かった。
いままでありがとう♡
